# Default Core Selector for Stock OS

This project brings the **Core Selector shortcut functionality** from [CrossMix-OS](https://github.com/cizia64/CrossMix-OS) to the stock operating system.

All credits to: cizia64 and his CrossMix-OS Project

It allows you to quickly set a default core emulator directly from the launcher for all games of a console or only to a specific game without needing the full CrossMix environment, making the stock OS much more lighter for users who want lightweight customization while keeping the original firmware.

---

## Features

- Adds Default Core Selector support to Stock OS with shortcuts
- Set default core for each game or for each console
- Add and update all available RetroArch cores and some standalones from CrossMix Brick tree.
- Easy to install

---
## Compatibility

Tested on:

- TrimUI Brick 
- Stock OS version on v1.1.0

Compatibility with modified stock environments may vary.

---

## Installation

1. Download the latest release
2. Extract the files to the root of your SD card
3. Reboot the device

After rebooting, the Core Selector shortcut should be available inside the launcher.

---

## Usage

1. Highlight a game
2. Open the Core Selector (Press Y button)
3. Highlight the emulator core you want to use
4. Follow the shorcut table:
<table align="center">
    <tr>
        <td><kbd>L</kbd> <sup>(hold) + <kbd>A</kbd><sup>(tap)</sup> </sup></td>
        <td>From Core selector: Set default core for a specific game and start</td>
    </tr>
    <tr>
        <td><kbd>R</kbd> <sup>(hold) + <kbd>A</kbd><sup>(tap)</sup> </sup></td>
        <td>From Core selector: Set default for the entire console system and start</td>
    </tr>
</table>

Note: A notification on the right corner shoud apper confirming the change

5. Launch the game normally

Your selected core will be used automatically.
